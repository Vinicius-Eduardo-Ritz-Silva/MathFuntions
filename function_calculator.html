Conceitos Matemáticos Fundamentais

Uma função é uma relação que associa cada elemento de um conjunto de entrada a um único elemento de um conjunto de saída.

Tipos de Funções:
  • Linear: f(x) = ax + b
  • Quadrática: f(x) = ax² + bx + c
  • Raíz Quadrada: f(x) = √x

Limite representa o valor que uma função se aproxima quando x tende a um determinado ponto.

Tipos de Aproximação:
  • Pela Direita (x → a⁺): Aproximação de valores maiores que a
  • Pela Esquerda (x → a⁻): Aproximação de valores menores que a
